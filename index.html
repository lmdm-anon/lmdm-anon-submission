Overview

This page hosts qualitative audio samples and anonymized clips from LMDM Generative Delay Interactions referenced in the submission. Replace placeholder assets by searching this file for REPLACE_AUDIO and REPLACE_IMAGE.

Abstract. Interactive streaming music generation promises generative models as true instruments. We investigate repurposing audio diffusion models into efficient interactive streaming models via block-wise KV caching and rollout-stable post-training (ARC-Forcing). We evaluate text-conditioned, sketch-conditioned, and accompaniment generation, including qualitative deployment as a real-time generative delay effect.

Accompaniment generation

Each row is a different accompaniment example. Columns vary the stem lookahead: how far into the future of the conditioning stem the model can see at inference time.

+2 s lookahead
The model sees 2 seconds ahead of the current playback position in the stem it accompanies.
0 s lookahead
The model sees the stem only up to the current playback position — no future information.
−2 s lookahead
The model sees the stem only up to 2 seconds before the current playback position — fully causal with an added buffer delay.

LMDM Generative Delay Interactions

Sessions used the real-time generative delay deployment described in the paper; anonymized screen captures are hosted on YouTube. If you cannot see the thumbnail, the link will take you to YouTube.

Guitarist demo with Jamendo trained LMDM (Enc-Dec, arc-forced, 230/10 context/block_size)

This clip shows a guitarist using the generative delay effect to play a song. This particular clip was generated with the prompt "EDM, strong sub bass groove, soaring synth leads". Notice how the delay effect transforms his guitar signal while maintaining a strong rhythmic structure the low end.

Saxophonist demo with Jamendo trained LMDM (Enc-Dec, arc-forced, 230/10 context/block_size, drum loop passed through sketch conditions for rhythmic consistency)

This clip shows a saxophonist using the generative delay effect. Same prompt as above, notice how as he learns that playing a particular triggers a deep sub bass punch, while also mimicking his playing with a synth lead as a delay, allowing him to create an orchestrated and dynamic mix on the fly.

Cellist with humpback whale calls trained LMDM (Enc-Dec, not arc-forced, 255/47 context/block_size)

This clip documents the first performance for the generative delay trained on humpback whale calls. The whale sounds start to come in at 2:43.
